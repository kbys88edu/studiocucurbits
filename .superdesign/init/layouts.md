# Layouts

## Page shell

- Source: `index.html`
- The page is a single `.site-shell` with a compact top bar, a hero, stacked editorial sections, and a footer.
- The desktop layout is a restrained off-white paper window; it becomes edge-to-edge on mobile.

## Shared layout source

All layout markup is in `index.html`. `styles.css` provides the grid, responsive breakpoints, border rules, and typography.
