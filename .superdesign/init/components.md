# Shared Components

This is a static HTML site. It has no separately authored component files; reusable UI is defined directly in `index.html` and styled by `styles.css`.

## Shared UI patterns

- `header.topbar`: studio mark, navigation, language controls, inquiry link.
- `.language-block`: Japanese, English, and French text variants, switched by `script.js`.
- `.primary-button`, `.secondary-button`, `.hero-link`: text-first studio CTAs.
